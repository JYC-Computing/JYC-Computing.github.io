正则表达式
用于匹配、搜索和处理文本的强大工具。它使用一种特定的语法来描述文本模式，并根据这些模式执行各种操作，如查找、替换、验证等。

创建正则
var reg = /abc/ //匹配字符串中abc字段

匹配
reg.test(str)，检测字符串str中是否包含reg规则，返回布尔值。 <script> var reg = /456/ var str = 'ghiaskjab123456' console.log(reg.test(str)) </script>

reg.exec(str) ，返回包含匹配结果的数组，没有则返回null <!-- 正则表达式对象的内置方法 --> <!-- 只返回第一个匹配结果 -->

str.match(reg) 返回包含匹配结果的数组 ,字符串内检索指定的值 <!-- 字符串对象的内置方法 -->

str.replace (reg,x) 将匹配到的内容用第二个参数替换 返回替换到的新字符串

str.split(reg) 按照正则规则切割

标志
修改匹配的行为 g 全局匹配 i 忽略大小写 <script> var reg = /hello/gi var str = 'hello world! HELLO th hello ere! hello!' console.log(str.match(reg)) </script>

转义
正则表达式 普通字符（abc123）和特殊字符(括号 运算符 ?^$)组成 . 表示匹配任意字符，即使在字符集里面，也表示匹配任意字符。

        var reg = /\./g
        var str = 'haha.safh'
        console.log(str.match(reg) )
字符集
匹配某一个范围的字符

[abc]：匹配字符集中的任意一个字符，即 a、b 或 c
[0-9]：匹配任意一个数字字符。
        var reg = /[0-9]/g
        var str = 'hah3a.bb2saf5cch'
        console.log(str.match(reg) )
[a-z]：匹配任意一个小写字母。

[A-Z]：匹配任意一个大写字母。

[\u4e00-\u9fa5]，匹配任一汉字字符。

[a-zA-Z]：匹配任意一个字母（包括大小写）。

[0-9a-z]，匹配 数字或者小写字母 字符。

\d 匹配任何数字 (阿拉伯数字)。相当于 [0-9]

\D 匹配任何非数字 (阿拉伯数字) 的字符。相当于[^0-9]

\s 匹配单个空白字符，包括空格、制表符、换页符、 换行符和其他 Unicode 空格

\S 匹配除空格以外的单个字符

\w 匹配基本拉丁字母中的任何字母数字字符，包括下划线

\W 匹配任何不是来自基本拉丁字母的单词字符。[^A-Za-z0-9_]

界定字符
. 表示匹配任意字符，即使在字符集里面，也表示匹配任意字符
^：匹配输入字符串的开头。
$：匹配输入字符串的结尾。
<script> var reg = /k$/ var str = 'akunkunk' console.log(str.match(reg)) </script>

断言 一种特殊的结构
/Hello(?=World)/
匹配Hello字符后面必须紧跟者World
        var reg = /Hello(?=World)/ 
        var str = 'dasdHelloWorlddasd'
        console.log(reg.test(str))
/(?<=Jack)Sprat/
它指定 "Sprat" 前面必须是 "Jack" 匹配 "Sprat"

/Jack(?!Sprat)/， 匹配字符串中的 "Jack"，但仅当它后面‌不是‌紧跟着 "Sprat" 时才匹配

/(?<!Jack)Sprat/ 确保在匹配 Sprat 之前，文本不应该是 Jack

(?=.*[A-Z]) 检查字符串中是否包含至少一个大写字母（A-Z）

量词
量词表示要匹配的字符或表达式的数量

量词普通写法 {n} n个 {n,m} n~m个，包含n也包含m {n,} n~无穷大个，包含n
<script> var str = 'hello xxx xx xuxx xxxx aaa' var reg = /x{1,}/g console.log(str.match(reg)) </script>

量词特殊写法

{1,} 可由 + 代替 匹配1到无穷(表示匹配1次或多次)

{0,} 可由 * 代替 匹配任意相同字符到无穷 其他字符转换成空字符 <script> var str = 'hello xxx xx xuxx xxxx aaa' var reg = /x*/g console.log(str.match(reg)) </script>

子项
定义正则时，可以用()将部分规则包裹起来， <script> var str = " tex123tTindexx1 text xxx"; var reg = /x(\w+)/g console.log(str.match(reg)) </script>

账号正则 ^[0-9]{5,}$：核心是 “纯数字 + 至少 5 位”，通过 ^/$ 确保无其他字符；

密码正则 ^[a-zA-Z][a-zA-Z0-9_]{5,16}$：核心是 “字母开头 + 后续 5-16 位字母 / 数字 / 下划线”，总长度 6-17 位；
