<table> 标记了表格的开始和结束 是所有表格相关元素的容器 <tr> 每个tr代表表格中的一行 <td> 代表表格中的一个单元格 用来显示数据 <th> 表头单元格 加粗和居中 区别普通单元格

<thead> 用于组织表头内容 包含一个或多个tr 通常包含th <tbody> 定义主体部分 包含多个tr <tfoot> 定义表格的底部

colspan合并列

rowspan合并行

border-spacing: 0px; 相邻单元格的边框之间的距离 border-collapse: collapse; 设置表格边框的折叠方式 collapse互相消融

表单
form表单用于收集用户输入并将数据提交到服务器 action 提交到哪个页面

get 提交的信息显示在页面的地址中 不安全 <!-- 搜索框 筛选条件 -->

post 数据不可见 更加安全 数据在请求体中(后端能接收) <!-- 登录 支付 -->

input
type text 文本框 password 密码框 submit 提交框 radio单选框

name 设置控件的名字

value设置控件的值

checked 默认

placeholder设置输入框的提示文本

autofocus开始就聚焦
