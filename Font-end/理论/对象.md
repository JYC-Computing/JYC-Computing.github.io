JavaScript 对象
对象(object):JavaScript里的一种数据类型 js对象拥有属性和方法的数据

某个车比喻对象 属性 颜色 黑 品牌 红旗 质量 1200kg 方法 启动 放音乐 加速 倒车

对象属性
<!-- 对象是变量的容器 --> 键值对的形式 一个对象可以有多个属性 <script> var person = { name:'坤坤', sex :'男', he:180 } </script>

访问对象属性
console.log(person.name) console.log(person['sex'])

var x = prompt() console.log(person.x)//person.'name' console.log(person[x])// person['name']

对象操作
对象本质无序的数据集合 增删改查

属性-改 对象名.属性=新值

属性-增 对象名.新属性=新值

属性-删 (了解) delete 对象名.属性

遍历对象
对象没有像数组一样的length属性 所有无法确定长度

for...in 循环 <script> var person = { name:'坤坤', sex :'男', he:180 } for(let i in person){ console.log(i) console.log(person.i) console.log(person[i]) } </script>
对象方法
对象的方法定义了一个函数，并作为对象的属性存储

    var person = {

内置对象
js内部提供的对象 包含各种属性和方法给开发者调用 <script> console.log(console) console.warn('你不能进入') console.clear() console.error('你代码有问题') </script>
