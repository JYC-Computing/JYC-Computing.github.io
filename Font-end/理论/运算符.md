比较特殊的两个赋值运算符自增1(++)、自减1(--)

自增运算符：双加号（++）是自增运算符，表示在“原来的值”的基础上再加上1。x++ 等价于x = x+1

x++ 指在使用x之后 再让x的值加上1

++x 指在使用x之前 先让x加上1

  var x = 6
  var y = ++x ;
  console.log(y)
  console.log(x)
自减运算符：双减号（--）是自减运算符，表示在“原来的值”的基础上再减去1。i--等价于i=i-1，自减运算符的使用同样也有以下两种情况。

i--指的是在使用i之后，再让i的值减去1。

--i指的是在使用i之前，先让i的值减去1。

JavaScript流程控制
流程控制，是任何一门编程语言都有的语法，指的是控制程序按照怎样的顺序执行。在JavaScript中，共有3种流程控制方式

顺序结构就是让程序按照从头到尾的顺序依次执行每一条代码，不重复执行任何代码，也不跳过任何代码。

选择结构也称分支结构，就是让程序“拐弯”，有选择性的执行代码。

循环结构就是让程序“杀个回马枪”，重复执行同一段代码。

分支结构
JavaScript中的分支结构主要有两种：if语句和switch语句。if语句通常用于实现单分支、双分支和多分支。

// 单分支: 当条件为true时，就会执行{}中的代码。小括号内的结果若不是布尔类型时，会发生隐式转换转为布尔类型。
if(条件){
	// 条件成立时执行
}
// 双分支：单分支的升级版，它不仅可以指定当条件成立要执行的代码，还可以指定当表达式不成立时要执行的代码。
if(条件){
	// 条件成立时执行
}else{
	// 条件不成立时选择执行
}

// 多分支：允许定义多个条件，并根据条件的结果执行相应的代码
if(条件1){
    //条件1成立时执行
}else if(条件2){
    //条件2成立时执行
}else if(条件3){
    //条件3成立时执行
}...else{
    //条件不成立时执行
}
三元运算符：允许你根据条件的真假选择两个不同的值。 condition ? expression_if_true : expression_if_false;

如果condition为真，则返回expression_if_true的值。
如果condition为假，则返回expression_if_false的值。
switch语句：用于值的匹配，case用于列出所有可能的值；只有switch()表达式的值与case的值匹配全等时，才会执行case对应的代码段。

default用来表示所有case都匹配失败的情况，一般写在末尾，做默认操作。

break用于结束匹配，不再向后执行；可以省略，break一旦省略，会从当前匹配到的case开始，向后执行所有的代码语句，直至结束或碰到break跳出。


        let day = 3
        let dayname;
        switch(day){
            case 1:
                dayname = '星期一';
                break;
            case 2:
                dayname = '星期二';
                break;
            case 3:
                dayname = '星期三';
                break;
            case 4:
                dayname = '星期四';
                break;
            case 5:
                dayname = '星期五';
                break;
            default:
                dayname = '周末放假 '
                break;
        }
        console.log(dayname)
循环结构
循环语句指的是在“满足某个条件下”循环反复地执行某些操作的语句。在JavaScript中，循环语句共有3种，while语句、do...while语句、for语句。

while (condition) {
    // 循环体（要重复执行的代码）
    // 注意：循环体内需要有改变循环条件的操作，以免陷入无限循环。
}
do...while循环是 JavaScript 中的另一种循环结构，与 while 循环不同的是， do...while 循环保证循环体内的代码至少执行一次，然后在每次迭代之前检查循环条件。

        do{
            x++
            console.log(x)
        }while(x<5)
for循环是 JavaScript 中最常用的循环结构之一，它允许你指定一个初始化表达式、循环条件和迭代表达式，然后在满足条件的情况下重复执行一段代码块。for 循环的基本语法如下：

for(定义循环变量;循环条件;更新循环变量){
	循环体;
}
break语句用于立即退出循环。当 break 语句被执行时，循环会立即终止，程序将跳出循环体，执行循环后的代码。

continue语句结束当次循环，开始下一次循环。
