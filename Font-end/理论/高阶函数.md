高阶函数
高阶函数可以接受函数作为参数(回调函数) 去操作数组

forEach‌：对数组的每个元素执行一次提供的函数。它不会返回新数组，只是用于遍历数组并执行一些操作
数组.forEach(function(当前元素, 索引) { // 对当前元素的处理逻辑 }); <script> var arr = ['fanfan',2,'坤','4',true] arr.forEach((x,y)=>{ console.log(x,y) }) </script>

map‌： 创建一个新数组，其结果是该数组中的每个元素都调用一个提供的函数后返回的结果。map不会改变原数组，而是返回一个与原数组长度相同的新数组
        var arr = [1,66,55,520,99]
        var x = arr.map((ele)=>{
            return ele*7.2
        })
        console.log(x)

filter 让你可以基于某些条件过滤数组，只保留满足条件的元素。
        var arr = [1,66,55,520,99]
        var x = arr.filter((ele)=>{
            return ele%2===0
        // 返回true 该元素被保留到新数组
        })
        console.log(x)
reduce对数组的每个元素执行指定的累积操作，返回最终的累积结果
array.reduce(function(累积器, 当前元素, 当前索引) { // 累积器返回上一次回调的返回值 }, 初始值);

some用于检测数组中是否至少有一个元素满足指定条件，返回一个布尔值
<script> const numbers=[1,3,5,7] const hasEven = numbers.some( num => num%2 ===0 ) console.log(hasEven) </script>

every测试一个数组内的所有元素是否都能通过指定函数的测试。它返回一个布尔值

  const todos = [
            { id: 1, text: '学习JS', completed: false },
            { id: 2, text: '写代码', completed: true },
            { id: 3, text: '复盘知识点', completed: true }
        ];
        const hasUncompleted = todos.every(todo => todo.completed )
        console.log(hasUncompleted)
