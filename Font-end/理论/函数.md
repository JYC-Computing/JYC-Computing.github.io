JavaScript函数
将一段代码打包在一起 每次调用函数 让这个代码块内的代码全部执行

两种情况下使用

重复使用的功能 函数很灵活 任意决定调用它的时机
特定的功能
函数声明
JS函数声明需要以function关键字开头，之后是创建的函数名称，function关键字与函数名称之间使用空格分开，函数名之后为一个括号( )，，后为一个花括号{ }，花括号中用来定义函数的函数体(即实现函数的代码)，如下所示：

function 函数名() { // 执行代码 }

调用函数
一旦定义好了一个函数，我们就可以在当前文档的任意位置来调用它。调用函数非常简单，只需要函数名后面加上一个括号即可。

myfunction()

onclick：javascript事件
onclick事件会在对象被点击时发生 <button onclick="a()">按钮</button>

调用带参数的函数
在调用函数时，您可以向其传递值，这些值被称为参数。这些参数可以在函数中使用。 您可以发送任意多的参数，由逗号 (,) 分隔：

函数声明的时候 小括号里面的是形参 形式上的参数 函数调用的时候 小括号里面的是实参 实际上的参数

arguments : 表示当前函数调用时传递的所有参数列表

<script> var x = 0; function a(age,sex){ // x++ // console.log(x) console.log('我的年龄'+age+'我的性别'+sex) console.log(arguments) } a(25,'男') </script>

带有返回值的函数
在使用 return 语句时，函数会停止执行，并返回指定的值。

    function a(age,sex){

            var text = '我的年龄'+age+'我的性别'+sex;
            return  text
        }
    console.log(a(25,'男'))
箭头函数

ES6 引入的一种更简洁的函数声明方式，使用箭头（=>）来定义函数。、
    var a = ()=>{
            console.log('你好')
        }
        a()
语法简洁 省略function关键字

当只有一个参数时候 省略括号

        var a = x => {
            console.log(x)
        }
        a('哈哈')
函数体只有一行 可省略return和大括号 var a = x => x console.log(a('哈哈'))

作用域
全局作用域 整个代码哪个地方都能访问 <script> var x = '全局' function aa(){ console.log(x) } aa() </script>

局部作用域 函数内声明的只有在函数内才能访问

    function aa(){
        var x = '局部'
    }
    aa()
    console.log(x)
