动画
在 CSS 中，@keyframes 规则用于定义动画的关键帧（keyframes），即动画在不同时间点的状态。通过指定元素在特定百分比（或 from/to）时的样式，可以创建平滑过渡的动画效果。

@keyframes move{ 0%{ width: 1500px; } 50%{ width: 800px; background-color: yellow; } 100%{ width: 500px; background-color: red; } }

@keyframes move { from{ transform: translate(0px);

        }
        to{
            transform: translate(500px);
            background-color: blue;
        }
    }
animation 属性
animation-name 执行哪一个关键帧动画 animation-duration 动画持续时间

animation-timing-function 指定动画变化曲线

ease 动画以低速开始，然后加速，最后在结束前减速

linear匀速

ease-in动画以低速开始，逐渐加速，直到结束。

ease-out 动画以高速开始，然后逐渐减速，直到结束。

ease-in-out动画开始和结束时都以低速运行

steps(3) 把动画分成了6步

animation-iteration-count 指定动画执行的次数

infinite 无线循环
animation-direction 指定方向

reverse 反方向
alternate 奇数次正向，偶数次反向
animation-fill-mode 动画最后保留哪一个值

forwards 最后一帧

backwards 动画第一帧的位置

animation-delay指定延迟执行的时间

animation-play-state: paused; 暂停动画

一起写 animation: move 2s 1s linear infinite alternate;
